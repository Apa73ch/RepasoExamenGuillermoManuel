🔹 2. Variables y Tipos de Datos

¿Qué es una variable?

Una variable es como una caja donde guardas información (datos) para usarla más tarde. Cada caja tiene un nombre.

nombre = "Adrián"
edad = 10
print(f"Hola, {nombre}. Tienes {edad} años.")

Tipos de Datos Básicos

Enteros (int)

Números completos, sin decimales.

Ej: 5, -20, 1000

Flotantes (float)

Números con decimales.

Ej: 3.14, -0.5, 99.99

Tipos de Datos Básicos

Cadenas (str)

Texto encerrado en comillas.

Ej: "Hola", 'Python'

Booleanos (bool)

Representan verdad o falsedad.

Solo pueden ser True o False

🔹 3. Tipos de Colecciones

Listas (list) 📝

Una colección ordenada y mutable (se puede cambiar). Permite duplicados.

frutas = ["manzana", "banana", "cereza"]
frutas.append("naranja")
print(frutas[0]) # manzana

Tipos de Colecciones

Tuplas (tuple) 📌

Una colección ordenada e inmutable (no se puede cambiar). Ideal para datos que no deben modificarse.

coordenadas = (10.0, 20.5)
print(coordenadas[1]) # 20.5

Tipos de Colecciones

Conjuntos (set) 🍇

Una colección no ordenada y sin elementos duplicados.

numeros_unicos = {1, 2, 3, 2, 1}
print(numeros_unicos) # {1, 2, 3}

Tipos de Colecciones

Diccionarios (dict) 📖

Una colección de pares clave-valor. No tienen un orden fijo (aunque en Python 3.7+ lo mantienen).

estudiante = {"nombre": "Ana", "edad": 9}
print(estudiante["nombre"]) # Ana

Tabla Comparativa de Colecciones

🔹 4. Estructuras de Control

Condicionales: if, elif, else

Permiten que tu código tome decisiones.

edad = 12
if edad >= 18:
    print("Eres mayor de edad.")
elif edad >= 13:
    print("Eres un adolescente.")
else:
    print("Eres un niño.")

Estructuras de Control

Bucles: for y while

Bucle for

Repite una acción para cada elemento de una secuencia.

for fruta in ["manzana", "pera"]:
    print(fruta)

Bucle while

Repite una acción mientras una condición sea True.

contador = 0
while contador < 3:
    print(contador)
    contador += 1

Palabras Clave en Bucles

break

Rompe y sale del bucle inmediatamente.

continue

Salta a la siguiente iteración del bucle.

pass

No hace nada. Es un marcador de posición.

🔹 5. Operadores y Prioridad

El orden en que Python resuelve las operaciones matemáticas.

  1. Paréntesis ()
  2. Potencia **
  3. Multiplicación *, División /, Módulo %
  4. Suma +, Resta -
resultado = 5 + 2 * 3 # Primero 2*3, luego + 5
print(resultado) # 11
